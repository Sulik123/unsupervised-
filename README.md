Что нужно иметь в папке
Файл с данными: Assignment-1_Data.csv (это тот, где колонки разделены точкой с запятой, а цены через запятую).

Сам скрипт первого задания (называй как хочешь, но у меня назван train_model.py).

Потом скрипты второго и третьего задания: Task_2.py, Task_3.py.

Папка templates с файлом index.html (для третьего задания).

1. Первое задание (обучение модели)
Запусти скрипт train_model.py. Он:

загрузит данные,

почистит,

сделает RFM и другие признаки (без квадратов, как мы договорились),

подберёт оптимальное количество кластеров (или ты сам поставишь 3),

нарисует график PCA и сохранит его как clusters_plot.png,

сохранит модели в файлы kmeans_model.pkl, scaler.pkl, pca_model.pkl.

Если после запуска в консоли видишь таблицу со средними значениями по кластерам – всё ок. Дальше эти модели будут использовать второе и третье задания.

2. Второе задание (консольное предсказание)
Запусти Task_2.py. По умолчанию откроется интерактивный режим:

Программа попросит ввести Recency (дней), Frequency (количество заказов), Monetary (сумму).

Вводишь числа (можно с точкой), нажимаешь Enter.

Получаешь ответ: «Кластер 0 — Активные постоянные» и т.д.

После одного предсказания цикл повторяется. Чтобы выйти, введи exit или q в любом поле.

Если нужно посмотреть статистику по всем клиентам сразу, запусти с флагом --all:

bash
python Task_2.py --all
Тогда программа выведет средние Recency, Frequency, Monetary для каждого кластера и размер кластеров.

Ошибки ввода (буквы, отрицательные числа) программа ловит и просит ввести заново. Ничего не падает.

3. Третье задание (веб-сайт на Flask)
Перед запуском убедись, что Flask установлен:

bash
pip install flask
Затем запусти Task_3.py. В консоли появится адрес типа http://127.0.0.1:5000. Открой этот адрес в браузере.

На странице:

Сверху написано, сколько всего клиентов и кластеров.

Ниже – график PCA (все точки, раскрашенные по кластерам).

Ещё ниже – форма для ввода Recency, Frequency, Monetary.

Вводишь данные, нажимаешь кнопку – под формой появляется результат: номер кластера и его описание.

Сайт работает локально. Чтобы остановить сервер, нажми в консоли Ctrl+C.

Настройка описаний кластеров
Если после обучения в первом задании средние значения по кластерам оказались другими (например, кластер 0 – не активные, а потерянные), просто замени словари LABELS в файлах Task_2.py и Task_3.py на свои. Текст описаний можешь написать любой, лишь бы номер кластера совпадал.

Если что-то пошло не так
Ошибка FileNotFoundError – проверь, что файлы с моделями и CSV лежат в одной папке со скриптами.

Ошибка ModuleNotFoundError – поставь нужную библиотеку: pip install pandas numpy matplotlib scikit-learn joblib flask.

График в третьем задании не рисуется – убедись, что в папке есть pca_model.pkl и что ты не удалял его после первого задания.

При запуске Task_3.py выдаёт TemplateNotFound – создай папку templates и положи туда index.html (код я давал).
